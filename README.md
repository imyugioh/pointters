# pointters-angular

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 1.5.0.

## Development server

```
  npm install
  npm start
```
Navigate to `http://localhost:4200/`

## Build

```
  npm run build
```
The build artifacts will be stored in the `dist/` directory.

## Running unit tests

```
  npm run test
```
